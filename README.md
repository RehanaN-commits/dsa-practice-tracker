# DSA Practice Tracker

A simple and visually clean frontend web application to track DSA problem-solving progress across different platforms.

## Why this project?
Students often practice DSA from multiple sources (LeetCode,HackerRank) 
This project provides a lightweight personal dashboard to track progress in one place.

## Features
- Add and track DSA problems
- Mark problems as solved
- Filter by solved and unsolved problems
- Visual progress bar
- Clean and colorful UI
- Data persistence using localStorage

## Tech Stack
- HTML
- CSS
- JavaScript

Built to demonstrate frontend fundamentals, UI design, and interactive logic.
